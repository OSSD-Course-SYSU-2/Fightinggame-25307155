# 格斗游戏 - 输了叫爸爸

一款基于HTML5 Canvas的双人格斗游戏，支持双人对战和人机对战模式。

## 📁 文件说明

- **原版-mimimi.html** - 游戏初始版本
- **升级版-mimimi-enhanced.html** - 经过优化和改进的版本（推荐使用）

## 🎮 游戏特色

### 核心玩法
- 双人格斗对战
- 人机对战（支持简单/普通/困难三种难度）
- 丰富的技能系统
- 灾难机制（烈焰、闪电、寒冬）
- 道具系统

### 技能系统
- **拳击攻击** - 基础攻击技能
- **闪避** - 快速后退躲避攻击（消耗20耐力）
- **推开** - 将敌人推开并使其短暂无法操作（消耗27耐力）
- **武器投掷** - 可投掷多种武器

### 灾难机制
游戏会周期性触发三种灾难：
- **烈焰灾难** - 地面出现火焰，造成持续伤害
- **闪电灾难** - 随机落雷，需要灵活躲避
- **寒冬灾难** - 召唤雪人发射雪球，被击中会被冰冻

### 道具系统
地图上会随机刷新道具：
- 回旋镖 - 可反弹的投掷武器
- 手雷 - 爆炸范围伤害
- 弓箭 - 高伤害远程武器
- 雪球 - 冰冻敌人

## 🎯 操作说明

### 玩家1（蓝色）
- **移动** - A/D 键
- **跳跃** - W 键
- **攻击** - F 键
- **防御** - S 键（按住）
- **闪避** - R 键
- **推开** - T 键
- **投掷武器** - G 键
- **调整角度** - 1/2/3 键

### 玩家2（红色）- 双人模式
- **移动** - 方向键 ←/→
- **跳跃** - 方向键 ↑
- **攻击** - 小键盘 1 键
- **防御** - 小键盘 2 键（按住）
- **闪避** - 小键盘 3 键
- **推开** - 小键盘 + 键
- **投掷武器** - 小键盘 0 键
- **调整角度** - 小键盘 7/8/9 键

### 通用操作
- **开始游戏** - 空格键
- **显示/隐藏按键提示** - H 键

## 🐛 已知问题

### 性能问题
- 大量粒子效果可能导致卡顿
- 长时间游戏可能积累内存

### 游戏平衡
- 冰冻时间较长（5秒）
- 灾难间隔较长（60秒）
- 部分道具过于强力

## 🎯 游戏建议

### 新手建议
1. 先在人机模式熟悉操作
2. 学会防御和闪避的时机
3. 注意灾难预警，提前躲避
4. 合理利用道具

### 进阶技巧
1. 推开技能可以打断敌人攻击
2. 闪避可以躲避灾难伤害
3. 冰冻时无法操作，要特别注意躲避雪球
4. 边缘尖刺可以用来击杀敌人
